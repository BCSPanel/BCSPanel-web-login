## 开始

下载源码并补全依赖

```
git clone https://github.com/BCSPanel/frontend-login
cd frontend-login
npm i
```

开发者实时预览

```
npm run dev
```

构建

```
npm run build
```

挂载预览（需要先运行构建）

```
npm run preview
```

构建并挂载预览

```
npm run bview
```
